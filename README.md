# Student-Performance-Prediction-System

## Overview
The Student Performance Prediction System is a Machine Learning application that predicts student academic performance based on various academic and behavioral factors.
The system analyzes attributes such as:
* Study Hours
* Attendance
* Learning Resources
* Motivation
* Online Courses
* Discussion Participation
* Assignment Completion
* Educational Technology Usage
* Stress Level

The model predicts student performance into one of four categories:
* Excellent
* Good
* Average
* Poor
